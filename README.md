## Playing "Being Lucky"

"Being Lucky" is played among 2 or more players, using 5 six-sided dice.


### Installation
```sh
$ git clone https://github.com/rokibulhassan/BeingLucky.git
$ cd BeingLucky
$ bundle install

### Play game
$ ruby play.rb

* Read prompt on terminal
* Press Y|n to continue or end Player turn

### Run Test

```sh
$ cd BeingLucky
$ rspec spec
